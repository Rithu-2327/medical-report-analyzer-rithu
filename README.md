**Medical Report Analyzer**

A web-based application that allows users to upload medical lab reports (PDF or image) and view extracted health data in a structured format.

**Overview**

This project extracts important medical parameters from uploaded reports and displays them clearly in a table. Users can review and manually edit values if required.

**Features**

Upload medical reports (PDF / Image)

Extract text from uploaded files

Parse key medical parameters

Display values in structured table format

Highlight abnormal values

Manual correction option

Basic diet suggestions based on selected parameters

Responsive UI

**Tech Stack**

React – For building the user interface

TypeScript – For type-safe development

Vite – For fast development and build setup

CSS / Tailwind CSS – For styling and responsive design

OCR / Text Processing Logic – For extracting and analyzing report data

Git & GitHub – For version control

Vercel – For deployment



**Installation**

git clone <your-repository-link>
cd <project-folder>
npm install
npm run dev

*Build*
npm run build
